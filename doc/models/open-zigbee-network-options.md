
# Open Zigbee Network Options

## Structure

`OpenZigbeeNetworkOptions`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `id` | `string` | Required | Multisense ID |
| `timeout` | `number` | Required | Amount of time in ms before closing the zigbee network again |

## Example (as JSON)

```json
{
  "id": "id0",
  "timeout": 202.56
}
```
